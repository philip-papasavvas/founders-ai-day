# Founders AI Day

A collection of projects and prototypes built during Founders AI Day at Dr Will's head office on Sunday 29 March 2026. The day brought together founders to explore ideas using AI tools, and this repo captures what was built.

We're publishing some of the ideas as static demos on Netlify. You can browse them at:

**https://founders-ai-day-demos.netlify.app/**

Navigate there to see the apps we built and static ideas from the day.

## Projects

| Directory | Description |
|-----------|-------------|
| `meal_planning_amp/` | Streamlit meal planning app for household weekly meals |
| `site/` | Static HTML demos deployed to Netlify |
